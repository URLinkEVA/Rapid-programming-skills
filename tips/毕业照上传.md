# 毕业照上传
参考[博客](https://blog.csdn.net/yijisama/article/details/120758073)

## 下载android studio
[下载](https://developer.android.google.cn/studio?hl=zh-cn)

下载完成安装记得勾选Android Virtual Device



## 卸载android studio

- 进入控制面板、卸载程序，右键卸载
- C:\Users\用户名目录下，删除文件夹：android, Android Studio3.5, gradle
- 删除sdk文件夹:Android
